Descrição Geral

O projeto_git é um site simples desenvolvido em HTML, focado em demonstrar conceitos básicos de estruturação web e conteúdo relacionado a receitas culinárias.
A proposta dessa atividade é entender como montar um projeto atraves de comandos do git e como fazer uma documentacao dos processos de desenvolvimento atraves dos commits.

Estrutura do Projeto

A estrutura inclui arquivos principais como index.html (página inicial com link para tags.html), tags.html (link para receitas.html), e receitas.html. Há subpastas para imagens, receitas específicas e resumos de tags HTML.
